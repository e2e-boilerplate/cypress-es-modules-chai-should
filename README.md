# Cypress Boilerplate
    
Cypress end-to-end test automation boilerplate, using ES Modules, Chai and Should.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/cypress-es-modules-should.git
    	2. cd cypress-es-modules-should
    	3. npm install
    	4. npm run test
