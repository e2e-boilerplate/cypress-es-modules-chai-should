  # Cypress Boilerplate
  Cypress end-to-end test automation boilerplate, using ES Modules, Chai and Should.
  ## Getting Started
  	 git clone git@github.com:e2e-boilerplate/cypress-es-modules-chai-should.git
    	 cd cypress-es-modules-chai-should
    	 npm install
    	 npm run test
